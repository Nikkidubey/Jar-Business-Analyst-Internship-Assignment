# 💼 Jar Business Analyst Assignment

## 📘 Overview
This project contains the complete solution for the **Business Analyst Internship Case Study – Jar**.  
It includes data analysis, visualization, and business insights prepared using **Python** and **Jupyter Notebook**.

---

## 🧠 Objectives
- Analyze company sales, profit, and target performance.  
- Identify top-performing and underperforming categories.  
- Explore regional sales patterns and improvement areas.  
- Suggest product and app enhancements for Jar.
